This template shows an example of how to deploy an Azure API Management service with custom hostnames.  This also demonstrates how to configure multiple proxy (gateway) custom hostnames for API Mangement service.  This template creates API Management service in Premium tier since the feature to configure multiple custom hostnames in proxy in API Management is only available in Premium tier of API Management.  However, you can configure single proxy custom hostname in any API Management tier.
