This template deploys a **simple Windows VM.**, using a few different options for the Windows Version, using the latest patched version.  
 
The parameters you gonna need to inform are **adminUsername** and **adminPassword**. This will deploy a Standard_D2_v2 size VM in the resource group location.

- [Azure Virtual Machines](https://azure.microsoft.com/services/virtual-machines/).
- [Azure Linux Virtual Machines documentation](https://docs.microsoft.com/azure/virtual-machines/linux/)
