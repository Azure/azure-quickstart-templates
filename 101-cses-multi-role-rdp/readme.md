This template allows to create cloud service (extended support) and also creates associated resources (virtual network, public IP address). 

Cloud Services (extended support) is a new Azure Resource Manager (ARM) based deployment model for Azure Cloud Services product and is currently in public preview. Cloud Services (extended support) has the primary benefit of providing regional resiliency along with feature parity with Azure Cloud Services deployed using Azure Service Manager (ASM). It also offers some ARM capabilities such as role-based access and control (RBAC), tags, policy and supports deployment templates.
