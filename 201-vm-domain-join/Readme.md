# Domain join sample

This template create a domain controller VM and a second VM that joins the VM over a private VM network connection.

This template deploys the following resources:

- Storage Account
- Virtual Network
- Public IP
- Load Balancer
- DC Virtual Machine
- Second Virtual Machine
