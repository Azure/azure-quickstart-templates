# Create a new VM on a new storage account from a custom image

<a href="https://portal.azure.com/#create/Microsoft.Template/uri/https%3A%2F%2Fraw.githubusercontent.com%2FAzure%2Fazure-quickstart-templates%2Fmaster%2F201-vm-domain-join-existing%2Fazuredeploy.json" target="_blank">
    <img src="http://azuredeploy.net/deploybutton.png"/>
</a>

This template allows you to join a Windows Virtual Machine into an Active Directory Domain.

Important to notice that the DomainJoinUserName parameter follows the domain\username format, where domain is the NetBios Domain Name.
