This template allows you to deploy a simple Windows VM using linked templates. The templates in this sample are created by modifying the template used in <a href="https://azure.microsoft.com/en-us/resources/templates/101-vm-simple-windows/">Deploy a simple Windows VM</a>. The intention is to show you how to create linked templates.
