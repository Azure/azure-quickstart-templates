This template creates a Media Services Account with its Storage account on Azure. The template uses the most recent **apiVersion** for **Microsoft.Media/mediaServices**. If you're creating an account in a government or sovereign cloud, you might need to use an earlier **apiVersion**, like "2018-07-01".

For more information about Azure Media Services, see [Media Services Documentation](https://docs.microsoft.com/en-us/azure/media-services/).
